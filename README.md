## Instrucción local

- 1: Crear archivo `.env` en la raíz del proyecto y agregar la siguiente variable

  ```bash
      VITE_BASE_URL=http://localhost:8080/api/v1
  ```

- 2: Instalar dependencias
  ```bash
    npm install
  ```
- Step 3: Iniciar el proyecto

  ```bash
    npm run dev
  ```
